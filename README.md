# 2026 성지순례 미사 · 묵주기도 지향

2026년 성지순례(6월 23일 ~ 7월 1일) 동안 봉헌되는 **미사 지향**과 **묵주기도 지향**을
날짜별로 확인하고, 봉헌자·지향 이름으로 검색할 수 있는 웹사이트입니다.

## 기능
- **오늘의 지향** — 그날 봉헌되는 미사 지향과 묵주기도 지향을 자동으로 표시
- **날짜별 보기** — 6/23 ~ 7/1 각 날짜의 순례지·미사 장소·시간과 봉헌 지향
- **검색** — 봉헌자 이름이나 지향에 적힌 이름으로 검색하면, 해당 지향이 언제 미사에서 봉헌되고
  어느 날 묵주기도로 봉헌되는지 보여줍니다.

## 지향 배분 원칙
- **미사 지향**: 문서에 따라 **파티마**(6/23·6/24)와 **루르드**(6/27·6/28·6/29) 미사에 봉헌.
  장소가 명시된 지향은 해당 장소에, 명시되지 않은 지향은 내용에 따라 배분.
- **묵주기도 지향**: 모든 지향이 순례 기간 중 최소 1회 이상 봉헌되도록 매일의 묵상 주제에 맞추어 배분
  (하나의 지향이 여러 날에 반복되어 봉헌될 수 있음).

## 구조
- `index.html` · `styles.css` · `app.js` — 화면과 로직
- `data.js` — 일정과 지향 데이터 (수정은 이 파일에서)
- `netlify.toml` — Netlify 배포 설정 (정적 사이트, 빌드 없음)

## 데이터 수정 방법
`data.js`의 `INTENTIONS` 배열에서 각 항목의 `massDate`(미사 봉헌일)와
`rosary`(묵주기도 봉헌일 목록)를 바꾸면 됩니다. 저장 후 다시 배포하면 반영됩니다.
